# 🛍️ Как запустить Витрину магазина (Frontend)

## 🎯 У вас есть 3 варианта:

---

## 🚀 **Вариант 1: Только Витрина** (рекомендуется для разработки)

Если вам нужна только витрина клиентов:

```batch
ЗАПУСК-ВИТРИНА.bat
```

**Откроется:**
- 🛍️ Витрина: http://localhost:3000

**Зачем:** Разработка фронтенда, тестирование UI

---

## 🎨 **Вариант 2: API + Админка** (текущий вариант)

Если нужны только бэкенд и админ-панель:

```batch
СТАРТ.bat
```

**Откроется:**
- 🔵 API: http://localhost:4000
- 🎨 Админка: http://localhost:3001

**Зачем:** Управление магазином, добавление товаров

---

## 🌟 **Вариант 3: ВСЁ ВМЕСТЕ** (полный запуск)

Если нужен весь проект сразу:

```batch
СТАРТ-ВСЁ.bat
```

**Откроется:**
- 🔵 API: http://localhost:4000
- 🎨 Админка: http://localhost:3001
- 🛍️ Витрина: http://localhost:3000

**Зачем:** Полное тестирование, демонстрация проекта

---

## 📊 Что такое Витрина (Frontend)?

### **Это клиентская часть магазина:**

✅ **Главная страница** - баннеры, акции, популярные товары  
✅ **Каталог товаров** - фильтры, сортировка, пагинация  
✅ **Страница товара** - фото, описание, характеристики  
✅ **Корзина** - добавление/удаление товаров  
✅ **Оформление заказа** - форма заказа, адрес доставки  
✅ **Личный кабинет** - история заказов, профиль  
✅ **Блог/Новости** - статьи о товарах  

### **Технологии:**
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Zustand (состояние)

## 🔗 Интеграция с API

### Витрина работает С API:

Если запущен API (http://localhost:4000):
- ✅ Товары загружаются из базы данных
- ✅ Корзина работает
- ✅ Оформление заказов работает
- ✅ Фильтры и поиск работают

### Витрина работает БЕЗ API:

Если API не запущен:
- ⚠️ Товары не загружаются
- ⚠️ Будут ошибки подключения
- ✅ Но UI всё равно можно посмотреть

**Рекомендация:** Запускайте API вместе с витриной!

## ⚙️ Настройка API URL

Витрина подключается к API по адресу:
```
http://localhost:4000/api/v1
```

Если ваш API на другом порту, измените в `.env.local`:

```env
NEXT_PUBLIC_API_URL=http://localhost:ВАШИ_ПОРТ/api/v1
```

## ❌ Проблемы и решения

### Проблема: Port 3000 уже занят

**Решение:**
```batch
# Остановите всё
ОСТАНОВИТЬ.bat

# Или убейте процесс на порту 3000
netstat -ano | findstr :3000
taskkill /F /PID [номер_процесса]
```

### Проблема: Ошибки при npm install

**Решение:**
```batch
cd apps\frontend
rmdir /s /q node_modules
del package-lock.json
npm install
```

### Проблема: Стили не загружаются

**Решение:**
```batch
cd apps\frontend
rmdir /s /q .next
npm run dev
```

Затем в браузере: **Ctrl+Shift+R**

### Проблема: Товары не загружаются

**Причина:** API не запущен

**Решение:**
```batch
# Запустите API отдельно
ЗАПУСК-API.bat

# Или всё вместе
СТАРТ-ВСЁ.bat
```
